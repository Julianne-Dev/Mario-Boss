<h1>Mario & Luigi: Interactive Experience</h1>
<p>Projeto responsivo desenvolvido para explorar o universo de Mario e Luigi através de uma interface moderna e interativa. O foco principal foi a aplicação de bibliotecas de ponta e boas práticas de desenvolvimento front-end.</p>

<h1>Tecnologias e Conceitos Aplicados:</h1>
*Three.js:* Integração de elementos 3D para uma experiência imersiva.

*JavaScript Moderno:* Lógica de interatividade e manipulação dinâmica de elementos.

*CSS Avançado:* Uso de Flexbox/Grid, animações e design responsivo (Mobile First).

*UX/UI Design:* Foco na experiência do usuário e em interfaces visualmente atraentes.

<p>Este projeto representa um marco importante na minha transição para o desenvolvimento tech, demonstrando minha capacidade de aprender novas bibliotecas e aplicá-las em soluções profissionais e divertidas.</p>
